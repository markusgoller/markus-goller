Weather Station

Markus | Sat 18 July 2020 | Updated on Thu 17 February 2022

As a studied atmospheric scientist it is almost a duty to operate a personal weather station (PWS).

Here you can see the outside sensors.

Below is a picture of the basis station.

It is a Renkfore WH2315 radio weather station. The station has a radio connection to a basis station and this is connected to a Raspberry Pi. I used Rasperry Pi OS (32-bit, released 2021-10-30). Via the open source software [WeeWX]((http://weewx.com/) the data is than hosted on Weather Underground.

Below you can see a my setting (arrows define the connections).

Here you can see a sample screenshot of my PWS taken from Weather Underground suitable to the photos above.

Installation of WeeWX and hosting via Weather Underground:

I followed basically the WeeWX debian documentation. 

pi@raspberrypi:~ $
wget -qO - https://weewx.com/apt/weewx-python3.list | sudo tee /etc/apt/sources.list.d/weewx.list
sudo apt-get update
sudo apt-get install weewx

Because the driver for the Renkforce WH2315 station is not available in WeeWX it has to be installed seperatly see https://github.com/EdwinGH/weewx-wh23xx/). I used than later WH23xx as the driver.

Basics of WeeWX:

Configurations are saved here:

pi@raspberrypi:~ $
/etc/weewx/weewx.conf

Quick change / initial setup in weewx.conf:

pi@raspberrypi:~ $ 
wee_config --reconfigure
wee_config --help

Commands:

pi@raspberrypi:~ $ 
sudo weewxd
sudo /etc/init.d/weewx status
sudo /etc/init.d/weewx stop
sudo /etc/init.d/weewx start
